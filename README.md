# AI-Vocational-Mechatronics

Open-source resources supporting the paper:
**AI-Based Control Integration into Vocational Mechatronics Education**.

## Included Files
- `nnpc_controller.py`: Neural-network predictive control (NNPC) demo with N=10 horizon.
- `flc_controller.py`: Mamdani fuzzy logic controller (FLC) demo.
- `example_dataset.csv`: Sample dataset used in experiment tasks.
- Licensed under MIT License.

## Usage
These codes are simplified teaching demos suitable for vocational training.
